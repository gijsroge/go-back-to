# Assets

Place your demo GIF here and name it `demo.gif`.

The GIF will be automatically referenced in the main README.

## Creating the GIF

You can create a GIF using tools like:
- [Kap](https://getkap.co/) (macOS)
- [Peek](https://github.com/phw/peek) (Linux)
- [ScreenToGif](https://www.screentogif.com/) (Windows)
- [LICEcap](https://www.cockos.com/licecap/) (Cross-platform)
- Online tools like [CloudConvert](https://cloudconvert.com/mp4-to-gif) or [EZGIF](https://ezgif.com/video-to-gif)

## Recommended Demo Flow

1. Show scrolling down on the homepage
2. Click a link to navigate to a detail page
3. Scroll down on the detail page
4. Navigate to a nested tab (e.g., `/detail/info`)
5. Click the back button
6. Show that it goes back to the homepage AND restores the scroll position
